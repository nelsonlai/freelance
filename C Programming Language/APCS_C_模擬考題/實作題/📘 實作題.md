📘 實作題 1 - 成績判斷 (Level 1)
📄 題目說明

輸入一個整數 score （0 ≤ score ≤ 100），判斷該成績的等第。

分數範圍	等第
90–100	A
80–89	B
70–79	C
60–69	D
0–59	F
📥 輸入格式
score

📤 輸出格式
輸出一個字元（A、B、C、D、F）
🧪 範例
輸入
85
輸出
B


📘 實作題 2 - 累加到 N（Level 1）
📄 題目說明
輸入一個正整數 N，請計算
1 + 2 + 3 + ... + N
📥 輸入格式
N

📤 輸出格式
輸出一個整數，代表總和。
🧪 範例
輸入
5

輸出
15

📘 實作題 3 - 找最大值（Level 1）
📄 題目說明

輸入三個整數 a b c，請輸出其中最大的數字。

📥 輸入格式
a b c

📤 輸出格式
輸出最大值。

🧪 範例
輸入
3 7 5

輸出
7

📘 實作題 4：奇數加總（Level 1）
📄 題目說明

輸入一個整數 N，請計算 1 到 N 之間所有奇數的總和。
📥 輸入格式
N

📤 輸出格式
輸出奇數總和。
🧪 範例

輸入
7
輸出
16
（1 + 3 + 5 + 7）

📘 實作題 5：數字反轉（Level 1）
📄 題目說明

輸入一個正整數 N，請將數字 反轉後輸出。

📥 輸入格式
N

📤 輸出格式
輸出反轉後的數字（不含前導 0）。
🧪 範例

輸入
12034

輸出
43021

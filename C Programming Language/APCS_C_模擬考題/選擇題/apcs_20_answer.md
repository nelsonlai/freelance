1. 正確答案：C（_sum）

📘 變數名稱規則：

可用英文字母、數字、底線

不能以數字開頭

不能使用保留字（如 class）

⚠️ class 是保留字，不能當變數名。

2. 正確答案：C（boolean）

📘 是否及格只有「是 / 否」，最適合用布林值。

⚠️ int 也能存 0/1，但語意不清楚。

3. 正確答案：A（2）

📘 整數除法只保留整數部分，捨去小數。

⚠️ 很多學生誤選 2.5（那是浮點除法）。

4. 正確答案：C（a = b）

📘

= 是指定

== 是比較

⚠️ 這是 APCS 超常考的陷阱。

5. 正確答案：B（main 函式）

📘 程式執行一定從 main() 開始。

6. 正確答案：C（string）

📘 string 是複合型別，不是基本型別。int, float, char, boolean

7. 正確答案：C（讓人閱讀程式）

📘 註解不影響程式執行，只幫助人理解。

8. 正確答案：C（&&）

📘

&&：AND

||：OR

!：NOT

9. 正確答案：B（程式無法執行）

📘 語法錯誤在編譯階段就會被攔下。

10. 正確答案：B（!=）

📘 多數程式語言中「不等於」都是 !=

📘 if 判斷一定是 true / false。

📘 else 只在 if 條件不成立時執行。

📘 switch 適合「多選一」情境。

15. 正確答案：B（B）

19. 正確答案：B（永遠不執行）

20. 正確答案：C（跳出迴圈）

📘 break 是用來結束最近一層迴圈。

三、迴圈（21–30）
21. 正確答案：B（固定次數）

📘 for 常用於已知次數的情況。

22. 正確答案：B（直到條件改變）

📘 while 常用於次數未知的狀況。

23. 正確答案：C（完全不執行）

📘 while 是「先判斷，再執行」。

24. 正確答案：A（條件永遠為 true）

📘 沒有出口 → 無限迴圈。

25. 正確答案：C（跳過本次迴圈）

📘 continue 會跳到下一輪。

26. 正確答案：B（5 次）

📘 i = 0,1,2,3,4 → 共 5 次。

27. 正確答案：B（忘記更新條件）

📘 這是造成無限迴圈最常見原因。

28. 正確答案：D（if）

📘 if 是判斷，不是迴圈。

29. 正確答案：D（break 結束迴圈）

📘 break：整個跳出
📘 continue：跳過一次

30. 正確答案：B（迴圈內有迴圈）

📘 巢狀迴圈常見於二維問題。

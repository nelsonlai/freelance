## üéØ Key Topics Coverage

### Excel Reading Operations (Week 2):
- ‚úÖ Read single sheets: `pd.read_excel('file.xlsx')`
- ‚úÖ Read multiple sheets: `sheet_name=None`
- ‚úÖ Read specific columns: `usecols=['A', 'B']`
- ‚úÖ Skip rows: `skiprows=3`
- ‚úÖ Handle headers: `header=None`, custom names
- ‚úÖ Data type handling: `dtype={'col': str}`
- ‚úÖ Cell-level reading with openpyxl

### Excel Writing Operations (Week 3):
- ‚úÖ Basic write: `df.to_excel('file.xlsx')`
- ‚úÖ Multi-sheet: `ExcelWriter` context manager
- ‚úÖ Append sheets: `mode='a'`
- ‚úÖ Cell-level writing with openpyxl
- ‚úÖ Formatting with xlsxwriter
- ‚úÖ Combined approach (pandas + openpyxl)

### Excel Merging Operations (Week 7):
- ‚úÖ Vertical merge: `pd.concat([df1, df2])`
- ‚úÖ Horizontal merge: `pd.merge(df1, df2)`
- ‚úÖ Join operations: inner, left, right, outer
- ‚úÖ Multiple file consolidation
- ‚úÖ Different structure handling

### Excel Insert Operations (Throughout):
- ‚úÖ Insert rows: `df.append()` or `pd.concat()`
- ‚úÖ Insert columns: `df['new'] = values`
- ‚úÖ Insert sheets: `ExcelWriter`
- ‚úÖ Insert formulas: `ws['A1'] = '=SUM(B:B)'`
- ‚úÖ Insert charts: `xlsxwriter` chart objects

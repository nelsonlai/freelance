## 快速導覽

### 🌊 Waterfall 瀑布式開發

**核心特點**：線性、序列式、階段性
- 適合需求明確、變更少的專案
- 強調文檔與規劃
- 每個階段完成後才進入下一階段

### 🏃 Agile/Scrum 敏捷開發

**核心特點**：迭代、增量、適應變化
- 適合需求變化快、創新型專案
- 強調協作與回饋
- 短週期迭代，持續交付價值

📢 JavaScript 輸出方法示範

1. console.log() - 主控台輸出

輸出到瀏覽器的開發者工具主控台，用於除錯。

console.log('這是主控台訊息');
💡 提示：按 F12 開啟開發者工具 → Console 標籤查看輸出

2. alert() - 彈出警告框

顯示一個彈出式警告框，會阻止頁面操作直到關閉。

alert('這是警告訊息');

3. document.write() - 寫入網頁

直接將內容寫入 HTML 文件。

document.write('這段文字會顯示在網頁上');
⚠️ 注意：在頁面載入後使用會覆蓋整個頁面內容！

4. innerHTML - 改變元素內容

修改特定 HTML 元素的內容。

document.getElementById('demo').innerHTML = '新內容';

這是原始內容，點擊上方按鈕改變我！

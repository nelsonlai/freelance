🎨 JavaScript 資料型別

1. Number（數字）

let age = 25;
let price = 19.99;
let negative = -10;

2. String（字串）

let name = '小明';
let message = "你好";
let template = `Hello, ${name}`;

3. Boolean（布林值）

let isStudent = true;
let hasLicense = false;

4. Object（物件）

let person = {
  name: '小明',
  age: 25
};

5. Array（陣列）

let colors = ['紅', '綠', '藍'];
let numbers = [1, 2, 3, 4, 5];

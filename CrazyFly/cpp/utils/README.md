# C++ Utilities

This directory contains utility functions for the CrazyFly quadrotor control system.

## Components

- `memory_manager.cpp` - Memory management utilities
- `thread_pool.cpp` - Multi-threading utilities
- `performance_optimizer.cpp` - Performance optimization tools

## Building Utilities

```bash
cd cpp/utils
make clean
make
```

## Features

- Efficient memory allocation and management
- Multi-threading support for parallel processing
- Performance monitoring and optimization
- Real-time system utilities

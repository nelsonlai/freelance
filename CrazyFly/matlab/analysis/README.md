# Analysis Tools

This directory contains analysis and optimization tools for the CrazyFly quadrotor system.

## Tools

- `parameter_optimizer.m` - Parameter optimization algorithms
- `performance_analyzer.m` - Performance analysis tools
- `data_visualization.m` - Data visualization utilities

## Usage

```matlab
% Run parameter optimization
optimizer = ParameterOptimizer();
optimizer.optimize_parameters();

% Analyze performance
analyzer = PerformanceAnalyzer();
analyzer.analyze_performance();
```

## Features

- Multiple optimization algorithms (GA, PSO, Bayesian)
- Comprehensive performance analysis
- Data visualization and plotting
- Statistical analysis tools

01 - Basic Canvas Setup

What is HTML Canvas?

HTML Canvas is an HTML element that provides a drawing surface for graphics. It's like a blank canvas where you can draw anything using JavaScript.

Key Concepts:

  • Canvas Element: The HTML <canvas> tag creates a drawing area
  • Context: The 2D rendering context is your "paintbrush" - you use it to draw
  • Coordinate System: (0,0) is at the top-left corner. X increases to the right, Y increases downward
  • Pixels: Everything is drawn in pixels

## Prerequisites

- **Python track**: Python 3.9+ (FastAPI, Flask)
- **Node track**: Node.js 18+ and npm (Node.js, Nest.js)
- **Databases**: MySQL/PostgreSQL for SQL; MongoDB for NoSQL (optional)

---

## Suggested Learning Path

1. **Start with REST** — Understand HTTP, methods, status codes, and resource design.
2. **Pick a language** — Python (FastAPI/Flask) or JavaScript/TypeScript (Node.js/Nest.js).
3. **Build CRUD APIs** — Apply REST principles with your chosen stack.
4. **Add persistence** — Integrate SQL or NoSQL based on your data model.
